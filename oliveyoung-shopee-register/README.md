# Olive Young → Shopee 당일 등록기

## 오늘 등록하는 순서
1. Shopee Seller Centre에서 첫 판매국의 최신 Mass Upload XLSX 템플릿을 내려받습니다.
2. 앱에 올리브영 URL을 한 줄에 하나씩 입력합니다.
3. 자동 추출 결과에서 원가·용량·무게를 확인합니다.
4. 판매국을 선택하고 등록자료를 생성합니다.
5. 상품명·가격·재고를 확인합니다.
6. 최신 Shopee 템플릿을 업로드하고 `쇼피 템플릿 자동 채우기`를 누릅니다.
7. 이미지 사용 권리를 확인한 뒤 1024px 이미지 ZIP을 받습니다.
8. Seller Centre에 XLSX와 이미지를 업로드합니다.

## 실행
```bash
cp .env.example .env.local
npm install
npm run dev
```
브라우저: http://localhost:3000

OpenAI API 키를 `.env.local`에 넣으면 국가별 AI 문구가 생성됩니다.
```env
OPENAI_API_KEY=...
```

## 제한
- Shopee 국가·카테고리별 템플릿의 필수 열은 서로 다릅니다. 앱은 상품명, 설명, 가격, 재고, 브랜드, 무게, 대표 이미지 URL 등 일반 열을 자동 감지합니다.
- 카테고리 ID, 배송 채널, 규제 속성 등 템플릿 고유 열은 첫 업로드 오류를 보고 직접 채워야 할 수 있습니다.
- 올리브영/브랜드 이미지는 사용 권한과 재판매 조건을 확인한 경우에만 사용하세요.

## 가격·할인 로직
- 기본 런칭 할인율: 20% (수정 가능)
- 기본 판매자 쿠폰 부담: 5% (수정 가능)
- 실판매가에서 플랫폼/결제 수수료, 광고비, 반품 충당, 쿠폰 부담 및 목표 순마진이 확보되도록 판매가를 역산
- 정상가는 `실판매가 ÷ (1-런칭 할인율)`로 계산
- 상품 화면에서 정상가와 할인가를 모두 수정 가능

## 자동 추정 후 수정 가능한 항목
- 카테고리명 / 카테고리 ID
- 제조사 / 원산지 / 상품 상태
- 사용기한(개월)
- 위험물 / 예약판매
- 발송 준비일 / 배송 채널
- 옵션명 / 옵션값

추정치는 등록 편의를 위한 초안입니다. Seller Centre 카테고리 선택 후 제공되는 공식 선택값과 제품 패키지 정보를 우선해 수정하세요.
